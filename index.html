Sécurité & Confidentialité

Vos données restent chez vous

Exécution 100% locale - Le script s'exécute entièrement dans votre navigateur. Aucun serveur intermédiaire.

Aucune collecte de données - Nous ne collectons, stockons ou transmettons aucune de vos informations.

Code source ouvert - Le code est entièrement visible et vérifiable sur GitHub.

Connexion directe - L'outil utilise votre session Wavy active. Vos identifiants ne sont jamais manipulés.

Comment ça marche 
// 1. Votre navigateur charge le script
fetch('exporter.js')

// 2. Le script appelle l'API Wavy
//    avec VOS cookies de session
fetch('api.wavy.fr/customers', {
  credentials: 'include'
})

// 3. Les données sont converties
//    en CSV localement
convertToCSV(data)

// 4. Le ZIP est créé dans
//    votre navigateur
zip.generateAsync()

// 5. Téléchargement direct
//    sur votre ordinateur
💾 export_wavy_20240115.zip

Questions fréquentes

Est-ce légal d'exporter mes données?
Oui. Conformément au RGPD (Article 20 - Droit à la portabilité), vous avez le droit de récupérer vos données personnelles dans un format structuré et lisible par machine. Cet outil vous aide simplement à exercer ce droit.
Pourquoi utiliser un bookmarklet?
Le bookmarklet permet d'exécuter le script directement dans le contexte de votre session Wavy. C'est la méthode la plus simple et la plus sécurisée car elle ne nécessite aucune extension ni partage de vos identifiants.
Les données sont-elles envoyées quelque part?
Non. Absolument aucune donnée ne quitte votre navigateur vers un serveur tiers. Les seules requêtes effectuées sont vers l'API officielle de Wavy (api.wavy.fr), et le ZIP est généré entièrement localement dans votre navigateur.
Ça ne fonctionne pas, que faire?
  1. Vérifiez que vous êtes bien connecté sur backoffice.wavy.fr
  2. Essayez de rafraîchir la page avant de cliquer sur le bookmarklet
  3. Désactivez temporairement votre bloqueur de publicités
  4. Essayez avec un autre navigateur (Chrome, Firefox, Edge)
  5. Ouvrez la console développeur (F12) pour voir les erreurs éventuelles
