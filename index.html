Questions fréquentes

Est-ce légal d'exporter mes données ?
Oui. Conformément au RGPD (Article 20 — Droit à la portabilité), vous avez le droit de récupérer vos données personnelles dans un format structuré et lisible par machine. Cet outil vous aide à exercer ce droit sur vos propres données.
Pourquoi un bookmarklet et non une extension ?
Le bookmarklet s'exécute dans le contexte de votre session Wavy active, sans aucune installation. Il est plus simple à auditer, ne nécessite pas d'accès aux permissions du navigateur, et fonctionne sur tous les navigateurs modernes.
Quelle différence entre CSV et JSON ?
CSV : données aplaties en tableau, directement ouvrable dans Excel ou LibreOffice. Idéal pour une utilisation métier.

JSON : données brutes avec leur structure imbriquée, idéal pour les développeurs ou pour importer dans un autre logiciel.
Ça ne fonctionne pas, que faire ?
  1. Vérifiez que vous êtes bien connecté sur backoffice.wavy.fr ou app.wavy.co
  2. Rafraîchissez la page avant de cliquer sur le bookmarklet
  3. Désactivez temporairement votre bloqueur de publicités
  4. Essayez avec un autre navigateur (Chrome, Firefox, Edge)
  5. Ouvrez la console développeur (F12) pour voir les erreurs éventuelles
  6. Signalez le problème sur GitHub avec le message d'erreur
